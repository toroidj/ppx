# Paper Plane xUI

ファイル操作用のツール集です。
大量のファイルを快適に操作するための機能と操作性に重点を置き、各ソフト間で処理を分担し、連携したファイル操作を行うことができます。

・PPb コンソールコマンドライン。クリップボード、ヒストリ、画面引用等の強力な編集機能と、コマンド実行を提供します。

・PPc ファイラ／ファイルマネージャ。以下の特徴を持っています。
	・１窓／独立表示２窓／一体・タブ付き複数窓／連結２窓等の表示
	・ファイルコピー・移動のバックグラウンド実行
	・拡張子やファイル内容等による独自判別実行
	・コンソールプログラムの実行結果をPPbで表示

・PPv マルチフォーマットビューア。テキスト／画像／16進ダンプ表示、S-JIS/JIS/EUC/ANSI/UNICODE等の文字コードを行単位で切替、HTMLタグの簡易解析等が可能です。

・PPtray ホットキー処理／ウィンドウ版コマンド実行を行います。

詳細は https://toro.d.dooo.jp/ppxhlp.html を参照してください。
バイナリは https://toroidj.github.io/slppx.html を参照してください。
